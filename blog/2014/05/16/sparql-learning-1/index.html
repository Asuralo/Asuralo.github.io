(II) Graph

*In this section, I understand graph by learning keywords FROM and FROM NAMED.

    (a)FROM builds only the default graph.

    (b)FROM NAMED builds only the named graphs.

    Like the psudo-code above, the portion GRAPH ?g{?s?p?o} means query all unnamed graphs. Thus, the code means get all triples in both the unnamed graph and named graphs.
